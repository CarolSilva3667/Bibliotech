## 📌 Sobre o Projeto

O projeto Bibliotech foi desenvolvido com o objetivo de modernizar e facilitar o acesso aos serviços de várias bibliotecas antigas, utilizando tecnologia para tornar a experiência dos usuários mais prática e eficiente.

A proposta envolve a criação de um sistema digital que permite a organização de acervos, consulta de livros, acesso a conteúdos e integração com plataformas online. Além disso, o projeto conta com protótipos, identidade visual e apresentação desenvolvidos em ferramentas como Canva e Figma.
